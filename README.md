# Lawrence Kofi Amoako | Graphic Designer Portfolio

Welcome to the official repository for my professional design portfolio. This project is built as a **Progressive Web App (PWA)**, allowing it to be installed directly to your device for a fast, offline-capable experience.

## 🚀 About the Project
This portfolio showcases my work in visual identity, digital production, and promotional design. It features an interactive "Workspace" sandbox, responsive design, and a modern aesthetic optimized for professional client presentations.

### Key Features
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile viewing.
- **PWA Ready:** Installable on mobile and desktop devices for offline access.
- **Interactive Sandbox:** Dynamic project staging area for portfolio management.
- **Theme Switching:** Seamless transition between Dark and Light modes.

## 🛠 Tech Stack
- **HTML5 & CSS3** (Modern Grid & Flexbox)
- **Vanilla JavaScript** (ES6+)
- **Service Workers** (for PWA offline functionality)
- **LocalStorage API** (for persistent project staging)
